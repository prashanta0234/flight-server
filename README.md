# Node.js Application

A simple flight booking application.

## Features
- Add Flights
- Watch Flight list
- Edit Flight
- Delete Flight
- Filter Flight
- Add seat booking with 2 min reservation
- Delete bookings
- Update bookings
- Login
- Registration

## Installation

1. Clone the repository:
   ```bash
   git clone [<repository-url>](https://github.com/prashanta0234/flight-server)
   cd flight-server
   ```
2. Run the project
   ```bash
   npm run dev / yarn start
   ```
